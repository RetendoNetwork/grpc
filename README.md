# grpc
- gRPC of NEX Server for Wii U & 3DS Consoles.

# Usage
- Install [Python](https://python.org/downloads/).
- Go to cmd, add to the cmd `pip install grpcio-tools`.
- Then, run the generate_grpc_files.sh or generate_grpc_files.bat.

# Installation
- Install [Python](https://python.org/downloads/) and [git](https://git-scm.com/downloads).
- Run the cmd, and add `pip install git+https://github.com/RetendoNetwork/grpc`
